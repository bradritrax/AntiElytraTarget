# AntiElytraTarget

**AntiElytraTarget**, Minecraft sunucularında elytra kullanan oyuncuların
haksız şekilde hedef alınmasını (target / aim / lock) engellemek için geliştirilmiş
hafif ve performans dostu bir plugindir.

## Özellikler
- Performans dostu (async / hafif event kullanımı)
- Kolay kurulum, ekstra ayar gerekmez
- Paper & Spigot uyumlu
- Açık kaynak

## Kurulum
1. **Releases** bölümünden en son `.jar` dosyasını indir
2. Dosyayı sunucunun `plugins/` klasörüne at
3. Sunucuyu **restart** et
4. Plugin otomatik olarak aktif olur 

## ⚙️ Gereksinimler
- **Minecraft:** 1.20+
- **Sunucu Yazılımı:** Paper / Spigot / Leaf / Folia
- **Java:** 17+
